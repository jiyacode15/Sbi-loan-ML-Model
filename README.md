# Sbi-loan-ML-Model
We built an ML-based loan default prediction system using ensemble models (LightGBM, XGBoost, CatBoost) with stacking to predict risk accurately. It provides risk scores and insights. A supporting SBI-inspired web portal (dupe version) enables real-time loan evaluation and decision-making.
Our project focuses on building an intelligent loan default risk prediction system using advanced machine learning techniques, designed to assist banks in making accurate and data-driven lending decisions. At the core of our system is a high-performance ensemble machine learning model that predicts the probability of loan default. We combined LightGBM, XGBoost, and CatBoost using a stacking approach, enabling higher accuracy, reduced variance, and better generalization compared to traditional single-model systems.
The model is trained on large-scale data (~300,000 applications) and enhanced through extensive feature engineering (300+ features) capturing key financial behaviors such as income stability, credit burden, repayment capacity, and risk indicators.
It is specifically optimized to handle class imbalance (~8% default cases) and evaluated using the ROC-AUC metric (~0.77+), which is widely used in real-world credit risk systems. 
Instead of giving a simple approval/rejection, the model generates a detailed risk analysis, including: Probability of default Risk category (low, moderate, high) Financial insights like EMI, FOIR, and recommended loan amount Intelligent recommendations for decision-making To support this system, we developed a web-based loan application portal that simulates a real banking environment. 
The website allows users to: Register and log in securely Fill structured loan application forms with dynamic validations Submit documents and financial details Track application status and raise complaints On the backend, the website acts as an interface layer, while the machine learning model functions as the decision engine, processing user inputs and generating real-time risk assessments through an interactive dashboard.
This integration demonstrates a complete end-to-end solution, combining machine learning with a practical deployment interface.
The system closely replicates how modern banking institutions evaluate loan applications, making it scalable, efficient, and aligned with real-world financial systems.
